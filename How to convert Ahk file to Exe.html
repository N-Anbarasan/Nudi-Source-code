How to make .exe file from AutoHotKey script of Nudi Kannada software?

Firstly, download AutoHotKey from its Website and install the same. After installing AutoHotKey, you will find a new application called Ahk2Exe.exe under AutoHotKey\Compiler\ in the folder/directory into which you have installed AutoHotKey. To create .exe file from .ahk file (to convert .ahk file to .exe file):

1. Run Ahk2Exe.exe by double clicking the “Convert .ahk to .exe” icon, you will get the following screen:

2. Click on Browse button for Source (script file) under Main Parameters section/group to select the AutoHotKey scipt .ahk file.

3. Now, Browse through the folder/directories and select the .ahk file for which you want to create an .exe executable file (select the .ahk file which you want to convert into an executable .exe file).

4. Click on Browse button for Destination (.exe file) under Main Parameters section/group to select the folder and to give name for the .exe file.

5. Now, Browse through the folder/directories where you want to save/store the .exe file created/converted for the .ahk file.

6. Type the name of .exe file, next to “File name” in the Folder/Directory opened and click on Save button.

7. Optionally, you may select an icon file for the .exe file, as you have selected the .ahk file by clicking on the Browse button.

8. Click on > Convert < button to create an .exe file for the .ahk file (to convert .ahk file into an .exe file).

9. The created / converted .exe file is saved in the folder/directory selected by you.

If you have included .ico while converting the .ahk file, the .exe will be displayed with that icon. Otherwise, it will be displayed with the AutoHotKey icon.
